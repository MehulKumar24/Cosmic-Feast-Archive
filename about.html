About the Project

About Cosmic Feast Archive

Cosmic Feast Archive is a creative HTML and CSS recipe website designed as a fantasy cookbook from the stars. It combines a futuristic interface with a magical food theme while keeping the layout clean, structured, and responsive.

Author

Mehul Kumar

Project Type

HTML & CSS Website

Theme

Fantasy meets science fiction

The visual style uses glowing borders, deep cosmic colors, elegant typography, and card-based layouts to present recipes in a way that feels both imaginative and easy to browse.

Purpose

Educational and creative showcase

This site was created as a web development project to demonstrate semantic HTML, CSS styling, responsive design, navigation, layout techniques, and consistent multi-page structure without JavaScript.

What it includes

Website features

  • A custom home page with featured recipe cards
  • Twelve recipe pages with images, ingredients, steps, time, and difficulty
  • Category archive pages for browsing grouped recipes
  • Shared navigation, footer links, and responsive styling across the site

Creative note

Original presentation

The recipe names and visual storytelling are intentionally themed to give the site a unique identity, while the underlying content remains simple, readable, and appropriate for an academic project.
